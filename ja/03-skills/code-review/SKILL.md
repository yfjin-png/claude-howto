# コード レビュー スキル

このスキルは、以下に焦点を当てた包括的なコード レビュー機能を提供します：

1. **セキュリティ分析**
   - 認証/承認の問題
   - データ露出リスク
   - インジェクション脆弱性
   - 暗号化の弱さ
   - 機密データのログ

2. **パフォーマンス レビュー**
   - アルゴリズム効率（Big O分析）
   - メモリ最適化
   - データベース クエリ最適化
   - キャッシング機会
   - 並行処理の問題

3. **コード品質**
   - SOLID原則
   - デザイン パターン
   - 命名規則
   - ドキュメンテーション
   - テスト カバレッジ

4. **保守性**
   - コード可読性
   - 関数サイズ（50行未満である必要があります）
   - 循環的複雑度
   - 依存性管理
   - タイプ セーフティ

## レビュー テンプレート

レビューされたコードごとに、以下を提供します：

### サマリー
- 全体的な品質評価（1-5）
- キー ファインディング数
- 推奨される優先領域

### クリティカルな問題（ある場合）
- **問題**: 明確な説明
- **場所**: ファイルと行番号
- **影響**: これが重要な理由
- **重大度**: Critical/High/Medium
- **修正**: コード例

### カテゴリー別の検出

#### セキュリティ（問題が見つかった場合）
セキュリティ脆弱性をサンプルで列挙します

#### パフォーマンス（問題が見つかった場合）
複雑度分析でパフォーマンスの問題をリストアップします

#### 品質（問題が見つかった場合）
リファクタリングの提案でコード品質の問題をリストアップします

#### 保守性（問題が見つかった場合）
改善でメンテナンス性の問題をリストアップします
