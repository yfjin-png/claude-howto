# API ドキュメントジェネレータ

API ドキュメントを生成します：

1. `/src/api/` 内のすべてのファイルをスキャン
2. 関数シグネチャと JSDoc コメントを抽出
3. エンドポイント/モジュール別に整理
4. マークダウンで作成
5. リクエスト/レスポンススキーマを含める
6. エラードキュメントを追加

出力形式：
- `/docs/api.md` 内のマークダウンファイル
- すべてのエンドポイントに curl 例を含める
- TypeScript タイプを追加
