# コード最適化

提供されたコードについて、以下の問題を優先度順に確認してください：

1. **パフォーマンスボトルネック** - O(n²) 操作、非効率なループを特定
2. **メモリリーク** - 解放されていないリソース、循環参照を検出
3. **アルゴリズム改善** - より良いアルゴリズムまたはデータ構造を提案
4. **キャッシング機会** - 繰り返される計算を特定
5. **並行性問題** - レースコンディションまたはスレッド処理の問題を検出

以下の形式で応答をフォーマットしてください：
- 問題の重大度 (重大/高/中/低)
- コード内の場所
- 説明
- コード例を含む推奨される修正
